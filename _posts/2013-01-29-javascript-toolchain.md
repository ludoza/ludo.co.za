# Javascript Development Toolchain

So recently I have been introduced to [node.js][node.js] and I started to explore what tools I can use to help me develop applications on my Mac(OS X). The most important part for me was debugging functionality and I was really existed after playing around the Cloud9 and Aptana.  
I also assume you can these applications under Windows and Linux. You might just struggle getting Cloud9 running on Windows, but I'm not sure.

##Cloud9

I have known about [Cloud9][Cloud9] for a while now. Cloud9 IDE is an open source web-based cloud integrated development environment that supports several programming languages, with a focus on the web stack (specifically JavaScript and NodeJS). It is written almost entirely in JavaScript, and uses node.js on the back-end. The editor component uses Ace.  
You can install a local version from their [github repository][cloud9github] and it support debugging for node.js it might also support debugging for Apache+PHP, Python and Ruby.

## Aptana

[Aptana Studio][Aptana Studio] is an open source integrated development environment (IDE) for building Ajax web applications. Based on Eclipse, it supports JavaScript, HTML, DOM and CSS with code-completion, outlining, JavaScript debugging, error and warning notifications and integrated documentation. Additional plugins allow Aptana Studio to support Ruby on Rails, PHP, Python, Perl, Adobe AIR, Apple iPhone and Nokia WRT (Web Runtime). Aptana Studio is available as a standalone on Windows, Mac OS X and Linux, or as a plugin for Eclipse.  
It has a nice Firefox/Firebug extension that allows you to debug javascript.   
I also know you can use firebug or chrome on its own. But it does not allow you to easily change your code and run it again, where I feel Aptana does a great job.  

## Mostly Everything Else

Finally for general text processing I use [TextMate][TextMate] for OS X, it is now open source, or you can look at [Notepad++][notepad] for Windows also open source.  
If you haven't figured it out by now I like to use open source tools.
