#Dropping Evernote

So tonight I stopped using Evernote(even if I have half a year of subscription left). Exported all my data the hard way and saved it to a git repo, to archive it online. 

Why you may ask? Because I feel evernote got slow, and also it did not allow me to do my note editing workflow as I would like to do it.

So now I have switch to having a directory on Dropbox containing my and accessing it with Textmate on my Mac and Nocs on my iPhone.

It does mean at the moment I'm giving up any search capabilities and it will hurt for a while. But I will be looking into the iPhone Elements Markdown Editor soon.

Btw as mentioned my new workflow is: Editing Markdown files on my pc using my text editor of choice allowing me to use markdown or plain text, Save it to git for archiving purposes and then share it with Dropbox.

I will see how it goes.
